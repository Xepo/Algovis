Visualization Help

Use sortinglist as your input. For visualization settings, place these somewhere in your code, and algovis will pick them up.
!vis-type: type;
vis-type sets the visualization type. Currently, the only supported selection is bar.
!vis-array: var;
var is a variable that contains the list to visualize.
!vis-index: var;
The index at var will be shown in a different color. This can be specified more than once.
!vis-indexrange: label var1 var2;
A range of bars will be sectioned off, between indices var1 and var2, with the label label.
!vis-extrabar: label var;
An extra bar will be drawn off to the side, with the label label, at the height var.
